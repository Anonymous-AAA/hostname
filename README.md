hostname
========


hostname provides commands which can be used to display the system's
DNS name, and to display or set its hostname or NIS domain name.

Reimplement the following commands in Rust:
* /bin/hostname
* /bin/dnsdomainname
* /bin/domainname
* /bin/nisdomainname
* /bin/ypdomainname
